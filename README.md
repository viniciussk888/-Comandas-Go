## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/viniciussk888/Comandas-Go.git
```

Follow the steps below

```bash
# Install the dependencies
$ yarn 

# Start the project
$ yarn dev
```
